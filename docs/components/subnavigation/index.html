Sub-navigation

Use the Sub-navigation component when there are pages available related to the chosen main navigation link (as indicated by aria-current="true" in the Header component above it). The class="ds-subnav" element is a <nav>, exposing it as a landmark region to screen readers. Note the use of aria-label="About". This differentiates this navigation landmark from the main one above it in screen reader landmark listings. Where the section is ‚ÄúOur work‚Äù, use aria-label="Our work".

The aria-current="true" attribution indicates the current page, in this case the first, index page.
