Markup

The basic Sidebar markup looks like the following. The elements do not need to be <div>s. Use semantic elements where appropriate. Note the ‚Äúinner wrapper‚Äù (an arbitrary div). This is used to help with the margin code.

<div class="ds-with-sidebar">
  <div> <!-- inner wrapper element -->
    <div class="ds-sidebar">
      <!-- sidebar content -->
    </div>
    <div class="ds-not-sidebar">
      <!-- not sidebar content -->
    </div>
  </div>
</div>

You can have the sidebar on the right instead, simply by switching the class="ds-sidebar" and class="ds-not-sidebar" elements over.
