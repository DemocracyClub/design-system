Accessible pairings

As the names suggest, the “forWhite” colors can be safely paired with $white and the “forBlack” colors can be safely paired with $black. For example:

.ds-my-component {
  color: $white;
  background-color: $greenForWhite;
}

.ds-my-component-reversed {
  color: $greenForWhite;
  background-color: $white;
}

The $pinkAccent color should not be used as text or as a background for text. It can be used for borders or other supplemental decorations.
